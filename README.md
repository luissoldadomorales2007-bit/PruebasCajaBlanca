# PruebasCajaBlanca
# Descripción del proyecto
Autoestopista Galáctico es un juego interactivo por consola inspirado en La Guía del Viajero Intergaláctico. El jugador lanza tres dados para generar una galaxia y avanza por casillas dependiendo de la cercanía entre galaxias. El objetivo es alcanzar la casilla 42, evitando peligros como agujeros negros o extraterrestres hostiles.

El juego incluye lógica de flujo condicional, bucles, generación aleatoria y control de estado, ideal para practicar estructuras básicas de programación en Java.

# Tecnologías utilizadas
Lenguaje: Java

IDE: IntelliJ 

Librerías:

java.util.Scanner para entrada por teclado

java.util.Random para generación de dados
